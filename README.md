A Python Discord bot that simplifies the usage of [Marzban](https://github.com/Gozargah/Marzban)'s Panel.

## Requirements
### Install uuid 
```
pip install uuid
```
### Install discord 
```
pip install discord
```
### Install marzpy 
```
pip install marzpy
```
